Stage all changes, write a conventional commit message based on the diff, commit, and push to origin.

Steps:
1. Run `git diff` and `git status` to understand what changed
2. Stage all changes with `git add -A`
3. Write a conventional commit message (type(scope): description) based on the actual diff — use feat, fix, docs, refactor, test, chore, etc.
4. Commit with that message
5. Push to origin
